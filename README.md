# Upload image api

| Methods | Urls             | Actions                                 |
|:--------|:-----------------|:----------------------------------------|
| POST    | /file            | upload a image                          |
| GET     | /file            | get a list of images (id, name and url) |
| GET     | /file/{name}     | get a image by name                     |
| GET     | /file/image/{id} | get a image by id                       |
